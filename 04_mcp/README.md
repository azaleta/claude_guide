# 第四章：MCP 模型上下文协议

MCP（Model Context Protocol）是 Anthropic 推出的开放标准，让 Claude 能够安全地连接各种外部数据源和服务。

---

## 什么是 MCP？

MCP 是大模型对其他资源或工具的统一访问接口标准，就像 AI 世界的 **USB-C**，让 Claude 能够：

- 访问本地文件系统
- 连接数据库
- 集成第三方服务（GitHub、Slack、Google Drive 等）
- 使用自定义工具
